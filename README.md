# YT-Downloader

**Description**  
this is a simple youtube video downloader written in python

## Requirements
### install yt-dlp
### for windows
```bash
pip install yt-dlp
```
### for linux
```bash
python3 -m pip install yt-dlp
```

## Installation

### clone repository
```bash
git clone https://github.com/SyntaxScripter/YT-Downloader.git

```
```bash
cd YT-Downloader
```
```bash
python main.py
```
